  • Face Detection and Recognition

    Utilizes OpenCV's cv2.CascadeClassifier for detecting faces in real-time from video streams or images. Detected faces are recognized using the Local Binary Patterns Histograms (LBPH) algorithm implemented by

    Face Detection: Detects faces using Haar cascades, a proven technique for real-time applications.

    Face Recognition: Employs the LBPH algorithm to identify individuals with high accuracy.

    Real-Time Performance: Efficiently processes video streams for immediate feedback.

  • Data Management and Training

    Organizes facial images into a dataset stored in a structured directory (DATASET_DIR). The system captures multiple images of each individual to enhance the accuracy of the recognition model.

    This data is then used to train the recognizer, which creates and stores the model in RECOGNIZER_DIR for later use.

    Dataset Management: Stores and organizes training images in a structured format.

    Training Process: Utilizes captured images to train the LBPH recognizer.

    Model Storage: Saves the trained model for efficient future recognition.

  • Secure and Encrypted Data Handling

    Ensures the privacy and security of facial images and recognition data by using secure storage methods.

    Data such as facial images and model parameters are stored in designated directories, with considerations for encryption and access controls to protect sensitive information.

    Secure Storage: Facial images and models are securely stored to prevent unauthorized access.

    Encryption: Uses encryption techniques to protect sensitive data.

    Access Control: Restricts access to data directories to authorized personnel only.

  • Web Integration and User Interface

    Integrates with Django to provide a web-based interface for managing attendance. Users can view attendance records, filter by date, and register new faces through the web application.

    This integration facilitates seamless interaction between the face recognition system and the user management interface.

    Django Integration: Provides a web-based interface for attendance management.

    User Interface: Allows users to view and manage attendance records through a web application.

    Real-Time Performance: Efficiently processes video streams for immediate feedback.
